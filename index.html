Performance Reports

These performance reports reflect the testing results for different thread configurations for two different versions of the WMS application. The tests measure the performance impact of running 20, 50, and 100 threads simultaneously. Each report contains details on system performance, including response times, throughput, and other relevant metrics.

The tests were conducted to measure response times for entity lists, such as containers, devices, alerts, and others, using partial filtering to simulate real-world platform usage scenarios.
